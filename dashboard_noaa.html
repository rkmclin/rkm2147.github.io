Snowfall General Trends

This graph is looking at the general trends of snowfall of at least 5 and at most 50 inches over the years.

noaa_df %>% 
  na.omit() %>% 
  filter(snow > 5 & snow < 50) %>% 
plot_ly(y = ~snow, color = ~year, type = "box", colors = "viridis" )

Maximum Temperatures vs Minimum Temperatures

This graph is a comparison of the maximum and minimum temperatures, which can be furthers specified based on the month of the year.

noaa_df %>% 
  na.omit() %>% 
  plot_ly(y = ~tmax, x = ~tmin, type = "scatter", mode = "markers", color = ~month, alpha = 0.5, colors = "viridis")

Snow Accumulation

This graph is a look at snow accumulation over the years. This can be further specified based on the station ID.

noaa_df %>%
  na.omit() %>% 
  filter(snwd > 0) %>% 
   plot_ly(y = ~snwd, x = ~year, type = "bar", color = ~id, colors = "viridis")
